This plugin will add a functionality to empty the cart of a woocommerce driven eShop when the visitor closes the browser.

== Description ==

This plugin clears the shopping cart of a woocommerce driven website when the visitor closes the browser. There are 3 
options for this plugin. 

*   Enable clear cart on browser closing
*   Enable clear cart on on user login
*   Enable clear cart on user logout

First one lets you to enable this dynamic cart clearing on your website.
Sencond and third options are here if for some strange reason you want to clear the carts if someone logs in or
logs out.

== Installation ==

Installation of this plugin is very simple.

1. Download and unzip the "wordpress-epdq.zip" file.
2. Upload the "wordpress-epdq" folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to  Settings > WC Clear Cart on Browser Close and set the parameters.
