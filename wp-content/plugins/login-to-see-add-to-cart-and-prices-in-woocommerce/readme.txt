Login to see add to cart and prices is useful if you want to hide add to cart buttons and prices in WooCommerce for not logged in users. You can also write your own personal message to invite the users to login and create a link to redirect users where you want. 


== Description ==

This plugin is an extension of WooCommerce and provide a simple settings panel where you can easily disable add to cart buttons and prices in your store for the users that are not logged in. You can also write your own message to invite the users to login or redirect users to the login page or to another page of your choice. 

= Features: =
* Disable Add to Cart buttons to every users that is not logged in
* Hide prices for every user that is not logged in
* Write your own alternative text that can be displayed in Add to Cart buttons or prices location both in archive page and single products page
* Select a page to redirect users where you want when they click on Add to Cart buttons.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. When the plugin is activated the settings are available in the WooCommerce menu > Login to see add to cart and prices. 




== Frequently Asked Questions ==

= Where are the settings of the plugins? =

The settings are very easy to find! The settings of the plugin are in the WooCommerce menu > Login to see add to cart and prices.


= Is the plugin tested for WordPress Multisite ? =

No, the plugin works for single ecommerce websites.
